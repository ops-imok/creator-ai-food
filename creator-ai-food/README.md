# Creator AI Food

> 实验性新菜创造网站 - AI 驱动的创意料理生成器

## 项目定位

**低成本实验性网站**：验证"新菜创造"概念

## 核心功能

### 一、新菜创造（主功能）

用户输入主食材，AI 生成创意菜品建议：

- 食材形态建议（块/片/丝/丁/整只/泥）
- 处理方式（煎/炒/炖/蒸/炸/卤）
- 搭配辅料/调料
- 创意亮点说明
- 可行性评分（1-5）

### 二、经典菜推荐（辅功能）

针对不知道做什么的用户：
- 按八大菜系分类推荐
- 提供 3-5 道固定经典菜
- 标准食材清单

### 三、社区评分系统

- 用户上传成品图
- 单维度评分（1-5 星）
- 动态展示规则：
  - ≥ 4.0：精选推荐
  - 3.0-4.0：正常展示
  - 2.0-3.0：待改进区
  - < 2.0：删除

## 技术栈

- 前端：HTML/CSS/JavaScript
- AI：百度千帆 / DeepSeek / 智谱 GLM
- 食材库：JSON（49 种常见食材）
- 图片存储：腾讯云 COS
