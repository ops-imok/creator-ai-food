# Creator AI Food

> AI-powered recipe creator - 实验性新菜创造网站

## 项目想法

这是一个基于 AI 的创意菜谱生成器，核心理念是：

**输入现有食材 → AI 创造全新菜谱**

## 核心功能规划

### Phase 1: 食材库 + 新菜创造界面
- 食材选择器
- AI 菜谱生成

### Phase 2: 经典菜推荐模块
- 基于食材推荐经典菜谱
- 烹饪步骤指导

### Phase 3: 用户上传 + 评分系统
- 用户分享自己的菜谱
- 社区评分机制

### Phase 4: 动态展示/降级逻辑
- 成品图展示
- 生成失败降级方案

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- AI API (百度千帆/智谱)
